# Contributing to ForensicGPT Framework

Thanks for considering a contribution. This project is a verification-grade forensic scaffold. Contributions must preserve:
- **Reproducibility**
- **Evidence integrity**
- **Auditability**
- **Misuse resistance**
- **Security-by-default**

Please read this fully before opening an issue or pull request.

---

## 1) Scope & Philosophy

### What this project is
- A **ChatGPT-like UI + API** to assist forensic workflows
- A **framework** for collecting evidence (Stage 1–3), analyzing (Stage 4), documenting (Stage 5), and presenting (Stage 6)
- A system where “AI text” is assistive, but **primary evidence artifacts** remain the source of truth

### What this project is NOT
- A tool to enable surveillance or unauthorized access
- A “self-fixing” system that changes itself without review
- A definitive biometric identification engine (voice/face identity proof)

---

## 2) Code of Conduct & Responsible Use
By participating, you agree to:
- `CODE_OF_CONDUCT.md`
- `RESPONSIBLE_USE.md`

We may reject contributions that significantly increase abuse potential.
